PART 4 — NAVIGATION
════════════════════════════════════════════════════════

In app/about.tsx, import router:
  import { router } from 'expo-router'

X button calls: router.back()

Make sure app/about.tsx is registered in the 
Expo Router file system. Since it is in the app/ 
root (not in app/(tabs)/), it will automatically 
be a stack screen accessible via router.push('/about').

If a stack navigator wraps it, set these options:
  headerShown: false  (we handle our own header)
  presentation: 'modal'  (slides up from bottom, 
                           feels natural from settings)
