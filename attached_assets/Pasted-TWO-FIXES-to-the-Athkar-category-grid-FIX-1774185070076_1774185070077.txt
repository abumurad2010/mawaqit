TWO FIXES to the Athkar category grid:

════════════════════════════════════════════════════════
FIX 1 — SPACING BETWEEN TILES
════════════════════════════════════════════════════════

Currently tiles are touching with no gap between them.
Add consistent spacing both horizontally and vertically.

In app/(tabs)/athkar.tsx, find the grid layout for 
the category tiles and apply:

GAP between tiles: 10px (both horizontal and vertical)
OUTER PADDING: 12px on left and right of the full grid

Recalculate tile size to account for gaps:
  const OUTER_PADDING = 12
  const GAP = 10
  const COLUMNS = 4
  const TILE_SIZE = (screenWidth - (OUTER_PADDING * 2) - (GAP * (COLUMNS - 1))) / COLUMNS

If using FlatList:
  columnWrapperStyle={{ gap: GAP }}
  contentContainerStyle={{ gap: GAP, paddingHorizontal: OUTER_PADDING }}

If using a manual grid (rows of Views):
  Each row: flexDirection row, gap: GAP, paddingHorizontal: OUTER_PADDING
  Between rows: marginBottom: GAP

Apply the SAME gap and padding values to the 
Favourites page grid so tiles are consistent.

Also apply the recalculated TILE_SIZE to placeholder 
tiles so they maintain the same spacing.

  Values: 'small', 'medium', 'large'
