The compass calibration button needs two changes:
1. When the calibration button is tapped, immediately perform a programmatic compass reset — restart or refresh the magnetometer/heading sensor so the reading is forced to update from the current sensor state. This gives the user instant feedback that something happened.
2. Remove any modal, alert, or popup that currently appears when the button is tapped explaining the figure-8 motion.
3. Below the calibration button, always display static instruction text — visible at all times, not only after tapping.
