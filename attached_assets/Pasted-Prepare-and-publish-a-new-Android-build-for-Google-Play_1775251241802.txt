Prepare and publish a new Android build for Google 
Play Store closed testing.

════════════════════════════════════════════════════════
STEP 1 — BUMP VERSION
════════════════════════════════════════════════════════

In app.json, update the following:

  "version": bump to next version 
    (e.g. 1.0.2 → 1.0.3 or whatever current is + 0.0.1)
  
  "android": {
    "versionCode": increment by 1
    (e.g. 7 → 8, or whatever current value + 1)
  }

Report the old and new values before proceeding.

════════════════════════════════════════════════════════
STEP 2 — COMMIT AND PUSH TO GITHUB
════════════════════════════════════════════════════════

Run in the Replit shell:

  git add -A
  git commit -m "Update to vX.X.X — thikr reminders, 
    default tab setting, Isha time fix, audio fix, 
    Quran offline bundle, widget support"
  git push origin main

Report success or any errors.

════════════════════════════════════════════════════════
STEP 3 — BUILD INSTRUCTIONS FOR JAMAL
════════════════════════════════════════════════════════

After pushing, print these exact instructions 
for Jamal to run on his Mac Terminal:

---
Run these commands on your Mac:

  cd mawaqit
  git pull origin main
  eas build --platform android --profile production

When the build completes (10-15 minutes), 
download the .aab file from the URL provided 
or run:

  eas submit --platform android --profile production

Then go to Google Play Console:
  → Mawaqit → Testing → Closed testing
  → Create new release
  → Upload the new .aab file
  → Add release notes (see below)
  → Review and publish
---
