ADD LONG-PRESS FAVOURITES to the Athkar category grid.

════════════════════════════════════════════════════════
BEHAVIOUR
════════════════════════════════════════════════════════

LONG PRESS on any category tile:
- Triggers a confirmation prompt asking:
  "Add [category name] to favourites?" 
  with two options: "Add" and "Cancel"
  (If already a favourite, prompt asks:
  "Remove [category name] from favourites?"
  with options: "Remove" and "Cancel")

ON CONFIRM — ADD:
- Tile immediately changes to the gold colour theme:
    • Border: #C9A84C (gold)
    • Background: #C9A84C at 10% opacity
    • Category name text: #C9A84C
    • Icon colour: #C9A84C
    • Small ⭐ badge in top-right corner of tile
- Tile moves to appear before non-favourited tiles 
  in the grid, maintaining relative order 
  among other favourites

ON CONFIRM — REMOVE:
- Tile reverts to default styling
- Tile returns to its original position 
  in the popularity-ranked order

PERSISTENCE:
- Store favourite category ids in AsyncStorage 
  under key 'athkar_favourites'
  as a JSON array: e.g. ["morning", "sleep", "istighfar"]
- Load on mount and apply gold styling immediately
- No maximum limit — user can favourite as many as they want
