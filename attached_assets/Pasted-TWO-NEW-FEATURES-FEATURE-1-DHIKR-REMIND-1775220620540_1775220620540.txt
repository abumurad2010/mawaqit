FEATURE 1 — DHIKR REMINDER NOTIFICATIONS
════════════════════════════════════════════════════════

WHAT IT DOES:
Sends silent banner notifications at random times 
throughout the day reminding the user to remember 
Allah. Each notification shows a random dhikr in 
the user's SELECTED APP LANGUAGE. No sound. No 
badge. Banner only. User can enable/disable in Settings.

  Line 1: Arabic text (always shown regardless of lang)
  Line 2: Translation in userLang (if lang is not Arabic)

  const getNotificationBody = (dhikr, lang) => {
    if (lang === 'ar') return dhikr.ar
    return `${dhikr.ar}\n${dhikr[lang] ?? dhikr.en}`
  }

This way Arabic-speaking users see only Arabic,
and all other users see Arabic + their language.

────────────────────────────────────────────────────
SCHEDULING LOGIC
────────────────────────────────────────────────────

Schedule 5 notifications per day at random times
between 30 minutes after Fajr and 30 minutes 
before Isha:

  const wakingStart = fajrTime + 30 * 60 * 1000
  const wakingEnd = ishaTime - 30 * 60 * 1000
  const wakingDuration = wakingEnd - wakingStart
  const interval = wakingDuration / 5

  let lastDhikrIndex = -1

  for (let i = 0; i < 5; i++) {
    const base = wakingStart + (interval * i)
    const randomOffset = Math.random() * interval * 0.8
    const notifTime = base + randomOffset

    // Pick random dhikr, never same as previous
    let index
    do {
      index = Math.floor(Math.random() * DHIKR_REMINDERS.length)
    } while (index === lastDhikrIndex)
    lastDhikrIndex = index

    const dhikr = DHIKR_REMINDERS[index]
    const lang = getCurrentAppLanguage()

    await Notifications.scheduleNotificationAsync({
      content: {
        title: t(dhikr_reminder_title),
        body: getNotificationBody(dhikr, lang),
        sound: false,
        badge: 0,
        priority: Notifications
          .AndroidNotificationPriority.DEFAULT,
      },
      trigger: { date: new Date(notifTime) },
    })
  }

RESCHEDULE:
  - On app open each day
  - When language changes — cancel and reschedule 
    with new language
  - When prayer times change (location/settings)
  - Cancel and reschedule if user changes language

────────────────────────────────────────────────────

AsyncStorage key: 'default_tab'
Default value: 'prayer-times'

In app/_layout.tsx, on app launch:

  useEffect(() => {
    const loadDefaultTab = async () => {
      const defaultTab = await AsyncStorage.getItem(
        'default_tab'
      ) ?? 'prayer-times'
      router.replace(`/(tabs)/${defaultTab}`)
    }
    loadDefaultTab()
  }, [])

Runs once on launch only.
Normal tab navigation is not affected after launch.
