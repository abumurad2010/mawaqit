CORRECTION 1 — SPELLING: dhikr → thikr everywhere
════════════════════════════════════════════════════════

Find and replace ALL instances of the following 
throughout the entire codebase including code, 
i18n keys, i18n values, comments, and file names:

  "dhikr" → "thikr"
  "Dhikr" → "Thikr"
  "DHIKR" → "THIKR"

This includes:
  - i18n keys: dhikr_reminder_title → thikr_reminder_title
  - i18n keys: dhikr_reminders_section → thikr_reminders_section
  - i18n keys: dhikr_reminders_label → thikr_reminders_label
  - i18n keys: dhikr_reminders_subtitle → thikr_reminders_subtitle
  - i18n keys: dhikr_reminders_activated → thikr_reminders_activated
  - AsyncStorage key: 'dhikr_reminders_enabled' → 'thikr_reminders_enabled'
  - Constant name: DHIKR_REMINDERS → THIKR_REMINDERS
  - File name: constants/dhikr-reminders.ts → constants/thikr-reminders.ts
  - Any variable names containing dhikr → thikr
  - Any display text in i18n.ts containing 
    "dhikr" or "Dhikr" → "thikr" or "Thikr"

Run this search to find all instances:
  grep -rn "dhikr\|Dhikr\|DHIKR" \
    app/ constants/ lib/ components/ \
    --include="*.ts" --include="*.tsx" \
    --include="*.js" --include="*.json"

Replace every single instance found.
