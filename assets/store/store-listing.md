## Full Description
Mawaqit (مواقيت) is a beautifully designed, fully offline Islamic app built for Muslims worldwide.

🕌 PRAYER TIMES
- Accurate prayer times for any city in the world
- 8 calculation methods: MWL, ISNA, Egypt, Makkah, Karachi, Tehran, Jafari, Gulf
- GPS-based auto-detection or manual city selection
- Adhan notifications for all 5 prayers
- Customisable Maghrib safety margin

🧭 QIBLA COMPASS
- Real-time compass pointing to the Kaaba
- Distance to Mecca displayed
- Works fully offline

📖 QURAN
- Complete Mushaf (604 pages) with authentic Uthmani script
- 14 translation languages
- Bookmarks and page memory
- Smooth offline browsing

🌙 ATHKAR
- Comprehensive morning and evening remembrances (Hisn al-Muslim)
- Ayat al-Kursi, Al-Mu'awwidhat, last two verses of Al-Baqarah
- Counter for each dhikr with haptic feedback
- Virtue (فضل) panel for each remembrance
- Daily notification reminders

📅 HIJRI CALENDAR
- Accurate Hijri/Gregorian calendar
- Islamic events and occasions highlighted

✨ DESIGNED FOR EVERYONE
- 15 interface languages including Arabic, English, French, Turkish, Urdu, Indonesian
- 7 colour themes including dark mode
- Clean, distraction-free interface
- 100% offline — no account, no ads, no tracking
